## ✅ Features

- Responsive layout with grid-based form structure
- Animated transitions and hover effects
- Input fields for:
  - First Name
  - Last Name
  - Email
  - Job Role (select)
  - Address (textarea)
  - City
  - Pincode
  - Starting Date
  - CV Upload
- Interactive button styling and focus effects

## 🌐 Technologies Used

- **HTML5**
- **CSS3** (Flexbox, Grid, Animations)
